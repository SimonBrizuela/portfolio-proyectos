# DataFlow Pro - Enterprise Data Management Platform

A professional, full-featured enterprise data management platform built with vanilla JavaScript, modern CSS, and semantic HTML. This application demonstrates advanced front-end development skills including data visualization, responsive design, and interactive UI components.

## Live Demo

Open `index.html` in your browser to see the application in action. No server or build process required.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/dataflow-pro.git
cd dataflow-pro
```

2. Open the application:
```bash
# Option 1: Open directly in browser
open index.html

# Option 2: Use a local server (recommended)
python -m http.server 8000
# or
npx serve
```

3. Navigate to `http://localhost:8000` if using a local server

## Usage

The application works entirely in the browser with no backend required. All data is stored locally using browser storage.

### Quick Start

1. Open the application in your browser
2. Navigate through different sections using the sidebar menu
3. Try the dark mode toggle in the top right corner
4. Add, edit, or delete transactions in the Dashboard
5. Export data to CSV format
6. Search and filter transactions in real-time
