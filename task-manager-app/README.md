# Task Manager Pro

Aplicación de gestión de tareas con interfaz moderna y funcionalidades avanzadas.

## Features

- Create, complete and delete tasks
- Priority system (High, Medium, Low)
- Multiple filters (All, Pending, Completed, High Priority)
- Real-time statistics
- Data persistence with LocalStorage
- Fully responsive design
- Modern interface with animations

## Technologies

- HTML5
- CSS3 (Flexbox, Grid, Gradients, Animations)
- JavaScript ES6+ (Classes, Arrow Functions, LocalStorage API)

## How to Use

1. Abre `index.html` en tu navegador
2. Escribe una tarea en el input
3. Selecciona la prioridad
4. Haz clic en "Agregar" o presiona Enter
5. Marca tareas como completadas haciendo clic en el checkbox
6. Usa los filtros para organizar tu vista
7. Elimina tareas individuales o en lote
